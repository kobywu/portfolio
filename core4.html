Loops
  • Loops in a type of code where you can repeat a set of actions as much as you want without writing it mulitple times
  • Real World Example: Instead of making one cake pop at a time, there is a pan where you can put 20 cake pops.
Condtionals
  • Conditionals are a code where you want something to happen only if something else happens. It is formated in "if this, then that."
  • Real World Example: If it is raining outside then I will bring an umbrella. Else I will wear shorts and a t shirt.
Variables
  • Variables are values that can be changed to anything.
  • Real World Example: In a game there is a score or a speed shown which is a variable.
Functions
  • If you want something to happen at different places or times in your code and you don't want to write it mulitple times, you can make a function which isn't on the main code but on the side. When the action is suppose to be called, you just call the function on the main code. This makes your main code look much cleaner and shorter.
  • Real World Example: If I am a cashier, when someone gives me money, I will give them back change then make the food.
